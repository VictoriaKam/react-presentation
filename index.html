The Secret of Popularity

  • Better Performance
  • Helps to prevent updating of DOM → faster apps, better UX
  • More total rendered pages from the website server

Argument against React

Frameworks are bad for application's health → better to use Web components instead

React Basics

  • component-based architecture
  • cohesive, reusable and modular components
  • display data as it changes over time

Component

independent and reusable bit of code

Collection of components → User Interface

Class Component

  • extends React.Component statement
  • render() method
  • must start with an upper case letter

Code Example:


	class MyComponent extends React.Component {
		render() {
			return (
				<elementOrComponent ../>
			);
		}
	}

Function Component

Code Example:


	const Hello = () => {
		return (
			<div className = "heading">
				<h1>Welcome to React!</h1>
			</div>
		)
	};
	ReactDom.render (
		<Hello />,
		document.getElementById("root")
	);

State

  • setState to change state
  • arrow function to bind

Code Example:


const Lake = ({name}) => <h1>{name}</h1>
class App extends React.Component {
	state = {
		loggedIn: true
	}
	render() {
		return (
			<div>
				The user is {this.state.loggedIn ?
				 'logged in' : 'not logged in'}
			</div>
		)
	}
}

Events

  • camelCase syntax
  • inside curly braces

Code Example:


	function ActionLink() {
		  function handleClick(e) {
		    e.preventDefault();
		    console.log('The link was clicked.');
		  }
		
		  return (
		    <a href="#" onClick={handleClick}>
		      Click me
		    </a>
		  );
		}
